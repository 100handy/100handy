# Supabase SQL Scripts

This folder contains ad-hoc SQL scripts for manual use in Supabase SQL Editor.

## How to run

1. Open Supabase Dashboard.
2. Go to SQL Editor.
3. Open one script from this folder and run it.

## Script categories

- `add_*` / `update_*`: Test data setup and updates.
- `fix_*`: One-off data/schema fixes.
- `check_*`: Diagnostics and validation queries.
